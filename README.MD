# Cool Admin Go Simple

## 使用

1. 安装依赖

```bash
make install
```

2. 运行

```bash
make dev
```

## 集成命令

Makefile 中集成了一些常用的命令，可以直接使用 `make` 命令执行。

```bash
build                          编译为二进制文件
cool-tools                     安装cool-tools
dev                            开发模式启动
help                           查看帮助
install                        安装依赖
```
